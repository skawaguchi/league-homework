To test the application, open the JavaScript console and access it at LeagueHomework.parseInput on the global window object.

Here is the example text from the instructions. You should be able to copy the text on the left of the equals sign and get the output text on the right.

'(9:00-10:00) "minus" (9:00-9:30)' = (9:30-10:00)
'(9:00-10:00) "minus" (9:00-10:00)' = ()
'(9:00-9:30) "minus" (9:30-15:00)' = (9:00-9:30)
'(9:00-9:30, 10:00-10:30) "minus" (9:15-10:15)' = (9:00-9:15, 10:15-10:30)
'(9:00-11:00, 13:00-15:00) "minus" (9:00-9:15, 10:00-10:15, 12:30-16:00)' = (9:15-10:00, 10:15-11:00)

For example:

window.LeagueHomework.parseInput('(9:00-10:00) "minus" (9:00-9:30)');

Should output:

"(9:30-10:00)"
